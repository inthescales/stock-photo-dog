# stock-photo-dog

Stock Photo Dog is a Twitter bot. It can be found at [@stock_photo_dog](www.twitter.com/stock_photo_dog).

Stock Photo Dog interacts with users who opt-in by mentioning it with the word "start". A user
can opt out by mentioning it again with "stop". It will follow any users who opt-in, and respond
to certain posts with a photo of a very enthusiastic dog.
